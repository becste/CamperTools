# Release Notes

## Version 2.9 (Build 20)

### New Features
- **3-Day Forecast:** Detailed daily forecast (Temperature, Precip, Wind) added to the "More data" screen.
- **Dual-Axis Calibration:** Independent Pitch and Roll calibration (in degrees) for precise leveling on any surface.
- **Auto-GPS:** Location is now automatically refreshed when the app opens or resumes.

### Improvements
- **Main Screen:** Now shows "Temperature" and current Wind Speed & Direction (e.g., "15 mph NE").
- **UI Updates:**
    - Shortened status error messages (e.g., "No location fix").
    - Improved layout for the Forecast screen with a sticky bottom "Close" button.
    - Consistent font sizing across data views.

## Version 2.8 (Build 19)

### New Features
- **Shake to Light:** The flashlight can now be toggled on/off by shaking the device (automatically enabled).
- **Enhanced Night Mode:** Night mode now turns all text and UI elements red to better preserve night vision.
- **Pitch & Roll Display:** Tilt indicator now explicitly labels "Pitch" and "Roll" angles.

### Improvements
- **Settings Redesign:** 
    - Reorganized settings into "Level Calibration" and "Units & Appearance" sections.
    - Renamed "Measure from current tilt" to "Auto Calibrate".
    - Renamed "Back" button to "Done".
- **Sun/Weather Detail:**
    - Renamed "Extra data" to "More data".
    - Renamed "Back" button to "Close".
    - Sunrise/Sunset times now respect the selected unit format (12h AM/PM for Imperial, 24h for Metric).
